Backup Manager

Automate backups of your Odoo databases with a simple, easy-to-use interface. Backups can be stored securely on:

  • Amazon S3
  • Nextcloud
  • Local server
  • Remote server via FTP or SFTP

Features include scheduled automatic backups, a backup history dashboard, one-click database restore, email notifications on success or failure, and role-based access control for restore and download operations.
